Markdown 從零開始 開發工具篇

2021/08/28

Markdown |

開始開發Markdown文件之前，首先我們需要一個好的工具，來幫助我們更友善的寫出一篇漂亮的Markdown文件，本篇將介紹使用免費又好用的編輯器 Visual studio code 來開發Markdown文件。

  • 下載安裝
  • Visual studio code編輯器

下載安裝

我們可以直接從 Visual studio code 官方網站，下載VSCode的安裝包(依照自己的開發環境選擇Windows/macOS/Linux)

Visual studio code編輯器

VSCode除了提供Markdown編輯，並且提供預覽以及快速檢視文件大綱。

Setp1. 建立Markdown文件

在工作區直接新增檔案，附檔名為 .md 的Markdown文件。

Setp2. 預覽Markdown文件

VSCode可默認正在編輯的是Markdown文件，在文件右上角的按钮點擊後，即可直接打開側邊的預覽功能，或是透過快速鍵 Ctrl + Shift + V 打開全視窗預覽模式。

Setp3. 視窗連動

當VSCode默認Markdown文件時，無論你是在編輯視窗介面或是預覽視窗介面，捲動滾輪，另一個介面就會同步連動你當前瀏覽的位置，這個功能也太棒了。

Setp4. 擴充套件安裝

套過VSCode工具列的Extensions可安裝第三方擴充套件(快速鍵 Ctrl + Shift + X)。

熟悉開發工具之後，快來動手試試看，寫一篇屬於你的Markdown文件吧 (Markdown 從零開始 基本文法篇)
