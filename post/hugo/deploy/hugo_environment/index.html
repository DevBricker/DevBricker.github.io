Hugo 從零開始 環境安裝(Windows)

2021/08/07

環境設定 |

創建我們的Blog第一步，怎麼就卡關了呢？很開心寫了腳本，想透過git直接更新我們的Blog，指令一下就GG。一起看看如何在Windows下打造我們的HUGO環境吧!

本篇將說明在使用Hugo建立Blog過程中遇到的環境問題解決方法，如何建立自己Hugo Blog請參考另一篇文章 (Hugo 從零開始 建立自己的部落格)

  • Hugo安裝(on Windows)
  • 環境變數

Hugo安裝(on Windows)

首先在本機端建立Hugo安裝路徑資料夾 C:\Hugo\bin 此路徑可隨自己慣用方式去建立，待會我們安裝Hugo時會用到。

參考官方網站釋出Hugo安裝檔

下載連結：Releases · gohugoio/hugo · GitHub

Setp1. 進入頁面後，向下捲動頁面，可尋找到最新釋出的安裝版本。

Setp2. 依照自己Windows系統，下載所需的版本，這邊我們使用的是Windows-64bit版本。

Setp3. 將下載下來的Hugo安裝壓縮檔進行解壓縮動作，這邊我們將解壓縮路徑指定於 C:\Hugo\bin 第一個步驟所產生的本機資料夾內。

環境變數

Setp1. 開啟cmd測試一下，將路徑指到C:\Hugo\bin底下，執行hugo version。

Setp2. 看起來好像成功了? 讓我們切換一下路徑，此時會發現找不到hugo指令可使用。

Setp3. 不要擔心，來設定一下開發環境的環境變數吧，找到變數Path點選編輯後，將我們的Hugo執行檔路徑C:\Hugo\bin新增進去。

Setp4. 再來一次，開啟cmd測試一下，在任何路徑底下，執行hugo version，成功。

Setp5. 回到專案終端機底下執行指令，成功。

解決環境問題之後，參考 (Hugo 從零開始 建立自己的部落格)文章，動手開始吧!
