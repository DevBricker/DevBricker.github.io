Visual studio code隱身在GitHub內的秘密

2021/08/21

Tool |

一個按鍵帶你發現隱身在GitHub內的秘密，快速開啟你的IDE(Integrated Development Environment)。

  • GitHub Hotkey
  • Demo

GitHub Hotkey

Step 1. 登入GitHub，進入任意一個Project的 GitHub Repository 畫面下 (不一定要自己的GitHub Repo，別人公開的Repo也可以唷)。

Step 2. 按下鍵盤快速鍵 【.】 (小數點)。

Step 3. 等候數秒，驚喜即將發生，GitHub將自動使用 GitHub Codespaces 開啟一個完全線上的 Visual Studio Code 開發環境。

透過GitHub Codespaces 支援線上 Visual Studio Code 可以在雲端進行開發、測試、版控，超級方便，快來試試看吧。

Demo

透過GitHub熱鍵按下鍵盤快速鍵 【.】 (小數點)，即可在雲端開啟屬於你的開發環境VSCode Visual Studio Code 線上編輯器。
